### Запуск проекта

1. Открыть проект в Unity (рекомендуемая версия: Unity 6)
2. Дождаться импорта пакетов (VContainer, UniTask и т.д.)
3. Открыть сцену: `Main.unity`
4. Нажать Play в Unity Editor

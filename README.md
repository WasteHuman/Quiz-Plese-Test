### Запуск проекта

1. Открыть проект в Unity (рекомендуемая версия: Unity 6)
2. Дождаться импорта пакетов (VContainer, UniTask и т.д.)
3. Открыть сцену: `Main.unity`
4. Нажать Play в Unity Editor

### Архитектура (кратко)
Проект построен на основе State Machine с разделением игровых состояний (Splash → Load → Gameplay и т.д.).

Каждое состояние реализует интерфейс IState и управляется через IStateController<GameState>.

DI реализован через VContainer.
Асинхронность — через UniTask.
Реактивность через R3.
Анимация прогресс-бара через DOTween.
